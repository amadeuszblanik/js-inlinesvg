## How to use
1. Install `js-inlinesvg` package. Eg. `yarn add js-inlinesvg`
2. Import `InlineSVG.js`. Eg. `import InlineSVG from "js-inlinesvg"`
3. Define InlineSVG in your `app.js` file. Eg. `const InlineSVG = new InlineSVG("inlineSVG")`
